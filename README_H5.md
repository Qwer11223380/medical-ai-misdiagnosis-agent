# 医学教育智能体 H5 原型

## 结论
可以通过 H5 方式调用免费模型实现一个可用的教学智能体原型。
本项目默认接入免 Key 的免费接口：`https://text.pollinations.ai/openai`。
当前版本为四幕锁步任务引擎：必须完成前一幕，下一幕才会解锁。

## 快速启动
在当前目录执行：

```bash
python3 -m http.server 8000
```

浏览器打开：

`http://localhost:8000/index.html`

## 使用流程（锁步）
1. 点击“加载目录资料”
2. 等待 PDF/DOCX/PNG 解析完成（图片会 OCR，可能较慢）
3. 进入第一幕并提交作答
4. 每幕通过后自动解锁下一幕（未通过只允许补答当前幕）

## 四幕结构
1. 第一幕：初诊导入
2. 第二幕：复诊反转
3. 第三幕：误诊复盘
4. 第四幕：迁移应用

## 解锁规则
1. 状态机固定顺序：M1 -> M2 -> M3 -> M4
2. 每幕有必答要点与最低分门槛
3. 不达标时系统返回缺失点和补答提示
4. 禁止跳幕

## 可替换模型接口
编辑 `app.js` 中 `askFreeModel` 函数的 URL 和请求体即可。
如果需要接入更稳定的免费额度（例如部分平台赠送额度），可使用带 Key 的 OpenAI 兼容接口。

## 说明
- 当前原型用于教学与比赛演示，不用于真实医疗决策。
- 免费接口可能有速率限制或偶发不可用，可在前端增加重试或切换备用接口。
